In order to run the website on your localhost, please follow the given steps:

1) Set up a webserver on your system if you don't already have one (Preferred: XAMPP).
2) Once XAMPP is successfully installed, run the XAMPP Control Panel and start the Apache and MySQL modules.
3) Navigate to the 'htdocs' folder within your XAMPP root and place the 'code' folder there. You can give it a different name if you wish.
4) Set up your MySQL database by opening localhost/phpmyadmin in your broswer window and creating a database called 'school'.
5) Then, import the school.zip file that has been provided.
6) Finally, navigate to the url http://localhost/code/public_html/index.php in your browser window to launch the website!
